# 🎓 StudyDesk — Student Productivity Dashboard

StudyDesk is a modern student productivity dashboard built with React.  
It helps students manage subjects, tasks, attendance, and GPA inside a clean and responsive interface.

The project focuses on learning React fundamentals, component-based architecture, state management, and dashboard UI design.

# ✨ Features

## 📚 Subject Management
- Add and remove subjects
- Store teacher names
- Add subject credits
- Organized subject cards
- Dynamic color indicators

---

## ✅ Task Manager
- Create study tasks
- Set due dates and priorities
- Assign tasks to subjects
- Mark tasks as completed
- Filter:
  - All
  - Active
  - Completed
- Overdue task indicators

---

## 📅 Attendance Tracking
- Track attendance records
- Attendance percentage calculation
- Subject-wise attendance analytics
- Interactive chart visualization

---

## 🎓 GPA Overview
- GPA tracking system
- Subject credit management
- Performance analytics
- Visual statistics dashboard

---

## 📊 Dashboard Analytics
- Subject statistics
- Task completion progress
- Attendance insights
- Current GPA overview
- Interactive charts using Recharts

---

## 🌙 Dark / Light Mode
- Custom theme system
- Persistent theme preference
- Modern dark dashboard UI

---

## 💾 LocalStorage Persistence
All data is stored locally inside the browser using LocalStorage.

- No backend required
- No database required
- Data remains after page refresh
